craft-atom-lock

2014-06-24 mindwind

简介

基于 redis 实现的一种分布式锁。

入门

该组件的一种典型使用场景如下：


DLock dlock = ...;
if (dlock.tryLock("lockKey", 10, TimeUnit.SECONDS)) {
    try {
        // manipulate protected state
    } finally {
        dlock.unlock();
    }
}

如何获得一个 DLock 实例


// base on redis 2.4+
DLock dlock = DLockFactory.newRedis24DLock(...);

// base on redis 2.6.12+
DLock dlock = DLockFactory.newRedis26DLock(...);
