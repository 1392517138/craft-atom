craft-atom

2014-06-18 mindwind

这是什么？

一个精心制作的 java 原子组件库。 基于 maven 的多模块项目，每个模块都是精心制作的提供特定功能的 java 原子库。 模块列表如下： 


1.  craft-atom-io
    reactor io 模型 api 定义。
2.  craft-atom-nio
    基于 java nio 的异步事件驱动网络 io 组件。
3.  craft-atom-redis
    基于 jedis 的易于使用的 java redis client。
4.  craft-atom-lock
    基于 redis 实现的一种分布式锁。
5.  craft-atom-util
    有用的工具类组件。
6.  craft-atom-test
    为其他原子组件库的单元测试提供支持的组件。
7.  craft-atom-protocol
    基础的协议编解码 api 定义。
8.  craft-atom-protocol-ssl
    简单的 ssl 协议实现者。
9.  craft-atom-protocol-http
    简化的 http 协议实现者。
10. craft-atom-protocol-textline
    简化的 textline 协议实现者。
