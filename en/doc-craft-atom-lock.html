Introduction

A kind of distributed lock base on redis.

Getting Started

A typical usage idiom for this component would be:


DLock dlock = ...;
if (dlock.tryLock("lockKey", 10, TimeUnit.SECONDS)) {
    try {
        // manipulate protected state
    } finally {
        dlock.unlock();
    }
}

How to get a DLock instance?


// base on redis 2.4+
DLock dlock = DLockFactory.newRedis24DLock(...);

// base on redis 2.6.12+
DLock dlock = DLockFactory.newRedis26DLock(...);
