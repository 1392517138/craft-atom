craft-atom

Feb 11, 2014 by mindwind

Introduction

A crafted and elegant atomic component library for java.

What is it?

It is a maven multi-module java projects, each module is a crafted and atomic java library for specific feature. The module list as follows: 


1.  craft-atom-io
    A reactor io api model definition.
2.  craft-atom-nio
    An asynchronous event-driven net io component base on java NIO.
3.  craft-atom-redis
    An easy use redis java client base on jedis.
4.  craft-atom-lock
    A kind of distributed lock base on redis.
5.  craft-atom-util
    A util component with some useful tools.
6.  craft-atom-test
    A unit test supported component used by other atom component.
7.  craft-atom-protocol
    A base protocol codec api definition.
8.  craft-atom-protocol-ssl
    A simple ssl protocol implementor.
9.  craft-atom-protocol-http
    A simple http protocol implementor.
10. craft-atom-protocol-textline
    A simple textline protocol implementor.
